## Что Это За Проект

KSD_AI — локальный Python-пайплайн для смыслового перевода Сири Гуру Грантх Сахиб на русский по методологии Karminder Singh Dhillon.

Основные слои:

1. `ksd_build_knowledge.py`
   Собирает `ksd_knowledge.db` из `.docx`, `.pdf` и `.txt` источников.

2. `ksd_ai_translator.py`
   Берёт ang JSON, группирует строки по шабдам, вытаскивает контекст из SQLite, отправляет промпты через Playwright в ChatGPT, парсит JSON-ответ и сохраняет результаты в `ksd_ang_json/` и `output/`.

3. `ksd_backup_db.py`
   Делает SQL-дамп и JSON-снапшот базы для версионирования и восстановления.

## Ключевые Артефакты

- `ksd_knowledge.db`: база знаний проекта
- `ksd_knowledge_dump.sql`: дамп SQLite для git
- `ksd_knowledge_snapshot.json`: краткий снапшот ключевых таблиц
- `ksd_ang_json/`: готовые KSD JSON-результаты
- `output/`: итоговые `.docx`
- `ksd_translator.progress`: прогресс длительного прогона
