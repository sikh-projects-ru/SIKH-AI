# Sri Guru Granth Sahib RU

Android reader prototype for the full Sri Guru Granth Sahib Russian corpus.

Build:

```bash
./gradlew assembleDebug
```

Content assets are generated by:

```bash
cd ..
python3 sggs_app/export_sggs_content.py
```

Current MVP:

- full offline SGGS corpus in assets;
- ang reader with previous/next and direct ang jump;
- drawer filters for works, raags, and authors;
- search within the current ang;
- visible layers: Gurmukhi, roman, Sahib Singh Punjabi, Sahib Singh Russian.
