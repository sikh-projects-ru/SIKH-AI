Текущая проблема: ChatGPT даёт слабый перевод в KSD-рамке.
Варианты решения (от лучшего к сложному):

### Вариант 1: Anthropic API (Claude) — РЕКОМЕНДУЕТСЯ
- Получить API-ключ на console.anthropic.com (pay-per-use)
- Переписать ksd_ai_translator.py под anthropic SDK (убрать Playwright полностью)

### Вариант 2: Улучшение промпта для ChatGPT
- Переключиться на GPT-4o (более умная модель)

### Вариант 3: Ollama + готовая модель (без дообучения)
- Установить Ollama, скачать llama3.1:8b или mistral

- Плюсы: бесплатно, офлайн
- Минусы: качество ниже Claude/GPT-4

### Вариант 4: Ollama + fine-tune на KSD-корпусе
- Набрать датасет: пары (gurmukhi + sahib_singh_pa) → ksd_translation

- Дообучить llama3 / mistral через Ollama или unsloth
- Требует GPU и ~50-100 переведённых шабдов для первого результата

### Вариант 5: Двухпроходный подход
- Проход 1: любой LLM генерирует черновик
- Проход 2: Claude/GPT-4 верифицирует по KSD-принципам и исправляет
