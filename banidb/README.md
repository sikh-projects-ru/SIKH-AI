# BaniDB — локальная база данных Гурбани

Инфраструктурный компонент проекта [SIKH-AI](https://github.com/sikh-projects-ru/SIKH-AI).

**Не переводческий инструмент — справочник и верификатор.**  
Единый источник истины по СГГС: другие системы проекта обращаются сюда для проверки целостности, поиска и сопоставления.

---

## Структура БД

Файл: `sggs.db` (SQLite)

Таблица `verses` — **14 261 строк**, все 1430 ангов СГГС.

| Колонка | Описание |
|---|---|
| `verse_id` | ID строки (из BaniDB API) |
| `shabad_id` | ID шабада |
| `ang` | Номер анга (страницы) = `с.` в цитатах |
| `line_no` | Номер строки внутри анга |
| `gurmukhi` | Текст на гурмукхи (unicode) |
| `transliteration` | Романизация (английская) |
| `translation_en` | Перевод на английский (BaniDB) |
| `translation_pu` | Перевод на панджаби (проф. Сахиб Сингх) |
| `writer_en` | Автор / Гуру (напр. `Guru Nanak Dev Ji`) |
| `raag_en` | Раг (напр. `Raag Aasaa`, `Siree Raag`) |

Индексы: по `ang` и `shabad_id`.

**Источник данных:** [api.banidb.com/v2](https://api.banidb.com/v2)

## Скрипты

### fetch_sggs.py — загрузка и обновление БД

```bash
# Скачать всё (анги 1–1430)
python3 banidb/fetch_sggs.py

# Продолжить с места остановки
python3 banidb/fetch_sggs.py --start 500

# Обновить один анг
python3 banidb/fetch_sggs.py --ang 418

# Принудительно перезаписать
python3 banidb/fetch_sggs.py --force

# Проверить статус БД
python3 banidb/fetch_sggs.py --status
```

### lookup.py — поиск из кода

```python
from banidb.lookup import SggsDB

with SggsDB() as db:
    # Все строки анга
    verses = db.get_ang(418)

    # Все строки шабада по ID
    verses = db.get_shabad(1610)

    # Поиск по английскому переводу
    results = db.search_en("weeping", limit=10)

    # Найти лучшее совпадение в анге по ключевым словам
    matches = db.find_best_match(ang=418, keywords=["weep", "cry", "mourn"])

    # Проверить, загружен ли анг
    ok = db.is_available(700)
```

## Как искать по цитатам

Цитаты в текстах проекта имеют формат `(Раг М1, с. NNN)`.  
Маппинг на поля БД:

| Часть цитаты | Поле БД | Пример |
|---|---|---|
| `с. 418` | `ang = 418` | страница = анг |
| `Аса` | `raag_en LIKE '%Aasaa%'` | раг |
| `М1` | `writer_en LIKE '%Nanak%'` | Махала 1 = Гуру Нанак |
| `М3` | `writer_en LIKE '%Amar Das%'` | Махала 3 = Гуру Амар Дас |
| `М5` | `writer_en LIKE '%Arjan%'` | Махала 5 = Гуру Арджан |

Пример запроса:
```python
with SggsDB() as db:
    verses = db.get_ang(418)
    aasa = [v for v in verses if "Aasaa" in (v["raag_en"] or "")]
```

## Назначение в проекте

- **Верификация ang_json** (`custom_khoj_sahib_singh`) — проверить, что переведённые строки соответствуют каноническому тексту
- **Поиск по цитатам** — автоматически подтягивать Гурмукхи + транслитерацию к цитатам из статей и книг
- **Проверка целостности** — убедиться, что в ang_json нет пропусков строк или битых ангов
- **Основа для будущих AI-инструментов** — единый справочник для KSD_AI и других систем
