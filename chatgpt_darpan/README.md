# ChatGPT Darpan Bot

Автоматически берёт страницы Guru Granth Sahib Darpan, отправляет в ChatGPT и сохраняет переводы в docx.

## Первый запуск

```bash
cd chatgpt_darpan
python bot.py --start 23 --end 50 --output ang_23-50.docx
```

При первом запуске откроется браузер. Если ChatGPT попросит войти — войди вручную,
потом нажми Enter в терминале. Сессия сохранится в папке `chatgpt_profile/`.

## Параметры

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `--start` | 1 | Начальный анг |
| `--end` | 1430 | Конечный анг |
| `--output` | `darpan_chatgpt.docx` | Имя выходного файла |
| `--delay` | 3.0 | Пауза между ангами (сек) |
| `--login` | — | Принудительно открыть страницу входа |

## Возобновление

Прогресс сохраняется автоматически в `.progress_<output>.txt`.
При повторном запуске с теми же параметрами скрипт продолжит с места остановки.

## Промпт

Промпт для ChatGPT настраивается в `bot.py` в переменной `PROMPT_TEMPLATE`.
